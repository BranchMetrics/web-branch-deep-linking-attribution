Web SDK delivery benchmark

Compares the two Branch Web SDK integration methods over repeated loads — the <script> tag (CDN) path vs the npm import (bundled inline) path. Each test runs branch.init() N times and aggregates mean / median / p95 / max plus First Contentful Paint and cache hit rate. Both exercise the SAME locally-built SDK, so the comparison isolates delivery.

💡 Tip: open DevTools → Network → throttle to "Slow 4G" before running, to see realistic numbers and the sequential _r → v1/open → v1/pageview waterfall. Each test opens in its own tab — run both and compare side by side.

<script> tag (CDN)

Classic <script src=…>. Loads the local build by default; add ?cdn=1 for the real CDN (load-only).
