EMT-3769 — npm import (bundled) delivery

Imports the SDK via a static import branch from 'branch-sdk', so the bundler (Vite/Rollup) inlines it into the entry chunk — the recommended npm delivery path. There is no separate SDK request, so total_init_ms and sdk_parse_ms are null by design (the SDK is compiled into the entry chunk and not separately attributable); the byte metrics describe the bundled chunk the browser downloads, and fcp_ms captures render time.
